# 프로젝트 2: 응급실 환자 도착 예측 (ED Arrival Prediction)

> **Agent:** Patient Flow Agent
> **예상 기간:** 4주 (Phase 2)

---

## 1. 목표
응급실 환자 도착 패턴을 시간대별로 예측하여 적정 의료진을 배치하고 대기 시간을 30% 단축한다.

## 2. 필요 데이터
| 데이터 | 출처 |
|:-------|:------|
| 응급실 내원 기록 (시간대/요일/KTAS) | ED 시스템 |
| 119 구급대 이송 데이터 | 119 |
| 지역 유동 인구/행사 정보 | 외부 |
| 독감/코로나 유행 데이터 | 질병관리청 |
| 기온/날씨 정보 | 기상청 |

## 3. 모델: LSTM + Prophet 앙상블

Prophet: 계절성/주기성 패턴 학습 (년/주/일)
LSTM: 단기 변동성 및 외부 변수 학습
앙상블: 가중치 결합 (Prophet 0.3 + LSTM 0.7)

## 4. 핵심 Feature
- 시간대별 도착 분포 (24h 프로파일)
- 요일 효과 (월~일)
- 계절성 (월별/분기별)
- 공휴일/특수일
- 기온/강수량
- 독감 유행 지수

## 5. 출력물
- 시간대별 예측 도착 환자 수 (KTAS 중증도별)
- 필요 의사/간호사 수 추천
- 약품/소모품 사전 준비 알림

## 6. 평가 지표
| 지표 | 목표 |
|:-----|:----:|
| MAPE | 15% 미만 |
| Peak-time MAPE (18-24시) | 20% 미만 |
| KTAS 1-2 예측 정확도 | 85% 이상 |
