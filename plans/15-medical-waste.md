# 프로젝트 15: 의료 폐기물 발생량 예측 및 처리 최적화

> **Agent:** Financial Agent
> **예상 기간:** 2주 (Phase 1 - Quick Win)

## 1. 목표
의료 활동 데이터 기반 폐기물 발생 예측, 처리 비용 20% 감소, 재활용률 15% 증가

## 2. 모델: ARIMA + XGBoost
- 폐기물 종류별(일반/감염/유해/재활용) 시계열 예측
- 외부 변수: 입원/외래/수술 환자 수, 검사 건수

## 3. 출력
- 일별/주별 폐기물 발생량 예측
- 처리 비용 추정
- 폐기물 수거 일정 최적화
